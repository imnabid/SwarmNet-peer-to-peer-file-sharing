**Get Started**
- Install rust, yarn

- Install tauri using yarn

- yarn tauri dev

- Devices must be connected on same network 

**What is it?**

-a p2p file sharing client like utorrent but within the same network

-inspired from bittorrent protocol

-not scalable as of now
